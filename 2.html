href 속성 : 하이러링크를 클릭했을 때 이동할 문서의 URL을 지정

target 속성 : 하이퍼링크를 클릭했을 때 현재 창, 새 창 여부 결정

 - _blank : 링크 내용이 새 창이나 새 탭에서 열림

 - _self : 현재 창에서 열림, target 속성의 기본 값

method 속성 : 서버쪽 프로그램으로 넘겨줄 방법을 지정

 - GET : 주소 표시줄에 사용자가 입력한 내용 표시

 - POST : 사용자가 입력한 내용시 표시되지 않음, 사용자의 입력을 표준 입력으로 전달

name 속성 : form의 이름을 지정, 구분자로 사용

action 속성 : form태그 안의 내용을 처리해 줄 URL을 지정



 태그는 type 속성에 따라 여러가지 형태로 표시

type="text" : 한 줄의 텍스트 입력

type="password" : 비밀번호 입력

type="date" : 사용자 지역을 기준으로 날짜(년, 월, 일) 입력

type="number" : 숫자를 조절할 수 있는 화살표

type="checkbox" : 주어진 항목에서 2개 이상 선택 가능한 체크박스

type="radio" : 주어진 항목에서 1개만 선택할 수 있는 라디오 버튼

type="submit" : 서버 전송 버튼

type="reset" : 리셋 버튼

type="button" : 기능이 없는 버튼

type="hidden" : 사용자에게는 보이지 않지만 서버로 넘겨지는 값

[일반 선택자의 종류]

전체 선택자 - HTML문서 내 모든 태그를 선택 ex) *{ }

타입 선택자 - 매칭되는 태그를 선택 ex) h1, h2, h3{ }

클래스 선택자 - class 속성 값과 매칭되는 태그를 선택 ex) .className{ }

ID 선택자 - id 속성 값과 매칭되는 태그를 선택 ex) #idName{ }

일반 선택자의 우선 순위 - ID 선택자 > 클래스 선택자 > 타입 선택자 > 전체 선택자



[박스모델]

content, padding, border, margin으로 구분

content : 화면에 표시되는 요소

padding : 박스 내부의 간격

margin : 다른 content(외부)와의 간격

border : content의 테두리



margin: 1px; // 모든 면에 적용

margin: 1px 2px; // 1px는 top, bottom, 2px는 left, right에 적용

margin: 1px 2px 3px; // 1px는 top, 2px는 left, right, 3px는 bottom에 적용

margin: 1px 2px 3px 4px; // 1px는 top, 2px는 right, 3px는 bottom, 4px는 left에 적용
